# AI-API ChatTool - Web Docker

一个创新的网页应用，让用户能够通过输入自己的API密钥，在网页上便捷地使用多种AI大模型服务。

## 项目特色

- 🚀 **便捷接入** - 只需输入API密钥，即刻体验AI大模型能力
- 🛠️ **灵活部署** - 支持Docker容器化部署，快速搭建专属AI服务平台
- 🌐 **网络优化** - 推荐搭配内网穿透技术，实现随时随地访问
- 💻 **友好界面** - 直观的网页交互设计，操作简单易上手

## 技术架构

本项目采用现代化的Web技术栈，结合Docker容器技术，为用户提供稳定可靠的AI服务接入平台。
